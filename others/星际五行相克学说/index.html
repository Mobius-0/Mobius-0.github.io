星际五行相克学说

一般情况下，战术能用上图的五个流派归纳。在双方实力相当的情况下，一般遵循莽克狗、狗克偷、偷克稳、稳克谐、谐克莽原则。 在目前的比赛中多边形选手居多，不再为单一风格

# 稳

—— 不动如山，动如雷震

稳，即稳健，防守型打法，早期以守为主，稳健平衡地发展科技、兵力和经济，求得在中后期时高人口决战取胜。代表人物：大部分职业星际选手，星际争霸时期的 FlaSh （最终兵器，猥琐神教教主） 、MacSed（胡翔） （怂神，万池王，momosed，sed 老师他不堵口） 、真实大帝 Reality、铁壁虫族 Soulkey、不动如山大雨神 Rain （最后变成了大狗神） 、打到后期的 Maru。

# 偷

—— 只要偷不死，就往死里偷

偷，即偷经济，最大化经济的打法，牺牲早期兵力，全力扩张以换取未来资源增长速度。代表人物：星际争霸 2 时期的 FlaSh （上天入地飞天遁地天下无敌世界无双宇宙第二无人第一烧酒蹭饭吃肉梅西最终兵器末日本座 start 狂魔肾大圣大胂大教主憋尿神功死亡冲锋 SCV 之友强行插眼全图开挂怒干跳跳屠鸡人族猥琐神教变鸡法爷运营如狗爆兵似马防奶神功提里奥运好弗丁刘欢李永菜灰烬使者大教主 flash 动画制作李永浩桑苏） 、以及教主真传弟子全教主 TY（农民比兵少算我输）。

# 狗

——DvD

狗，即阴险、高风险、非常规的赌博性打法，一般用来形容野外藏生产或科技建筑、地堡 rush 等战术。代表人物：Slayer_BoxeR （上古大神，初代本座，众狗之祖） 、sOs （狗哥，天下三狗之首，匪夷所思狗，绿翼双犬之一，30 万美金的男人，OOO） 、Jim（曹晋珲） （天下三狗之一，霸王硬上狗，修地堡天赋 MAX） 、Has （天下三狗之一，台湾神皇，天马流星狗，在欧美被称作 Yoe 家族 Has 一世，水晶墙之主，最激进的野建筑，最具洞察力的先知，东方巨狼，Kaelaris 的破坏者（Kaelaris 是一个外国解说，有一次解说 Has 比赛的时候笑到抽筋），暴君驯服者，InCa 之子（InCa，在 GSL 决赛连续放了雀茶四把隐刀然后被横扫），sOs 之兄，聚焦一切光辉之棱镜） 、Maru （和狗哥一个卵里出来的双犬之一） 、绿翼第三犬 TRUE Cure Trap Rogue 等人。 莽霸去了绿翼都变狗了，这就是近狗者狗吗？

# 莽

—— 我叫钱赞企！！！

莽：即莽夫，极限进攻型打法，中前期有一定经济后，牺牲未来的发展，投入所有经济疯狂暴兵进攻，力求短时间内迅速打垮对手。代表人物：Courage（钱赞企） （蟑螂，NO 总） 、枪兵王 MarineKing （山包，红粉娘娘） 、总统 MC （欧美捞钱王） 、Toodming（土豆明）、吕布 Innovation、输本 soO、莽霸 TRUE、不修地堡时的 Jim、TIME（李培楠） （凯子人皇，野人，李老八） 。

# 谐

—— 遭不住了啊 xdm

谐，即谐星，不以胜利为第一要务，而是旨在彰显智商优越的观赏性 侮辱性 战术，往往在低水平竞技中能发挥奇效。代表人物：F91（孙一峰） （智障也有春天，谐会会长，说了算孙哥） 、小色（黄旭东） （炸屎人皇，专业毒奶） 、XY（鸟哥） （夕阳红人族，XY U Suck） 、Parting（胖丁，跳跳、 王丽萨 ）。 其中 “稳”、“莽” 一般是比赛中的主流。

# 奶

—— 干死黄旭东，巨像能对空

在稳、偷、狗、莽、谐之上，有唯一神 “奶”，奶克一切。 宝矿力能解解毒，减缓毒奶发作的时间与减小威力，然而最终还是要被毒死。旭东要奶死，最终还是要死的，“六君子” 音容犹在。

谈谈《龙族》动画

发表于 2022-11-19 本文字数 1.2k 字 阅读时长 1 分钟
  • FAF MAVE
首页 杂谈 影视作品

终于，江南这个狗贼还是给我们整了个大活，整挺好，就是有点稀烂。

我强迫自己看完了前三集，最后还是没有勇气继续看下去，因为就前三集而言，动画本身除了没有像其他有的 IP 改编大刀阔斧的重置剧情以外，感觉处处是缺点。

首先先表扬一下优点，剧情没有魔改，即使有一点小改动但也无伤大雅，剧情的大体方向是相当还原的。人物的设计还算不错，至少我个人看的还算舒服，没有那么出戏。

好了，夸完之后接下来来谈一谈缺点，缺点可以说很多啊，配音、细节还原、节奏安排等等都是问题。首先开幕雷击啊，不闪不避直接上来就给我一个普通的直拳，原作中路明非是在玩星际一，动画中画面一转玩的是什么原创游戏，星际一里面有围墙吗？有超武吗？好端端一个 2.5D 的 RTS 游戏被画成平面 2D 你好意思吗？你这游戏画面里一个兵都没有全是围墙建筑打的是塔防吗？你音乐都请了泽野弘之都不舍得买个游戏体验一下的吗？星际一的重制版出了也有一段时间了你都不舍得管网易暴雪买一份要个授权啥的吗？ 哦 zz 暴雪要中止合作啊，那没事了，暴雪好似开香槟！！ 刚刚开始看就给我一个大逼兜，接下来要整什么花活我都不敢想。

所幸接下来的花活都很烂，越往下看越发觉配音问题很大，总有点棒读或者尬念的感觉，古德里安教授的配音我就没有绷住，虽然人对路明非确实很感兴趣，对这么一个学生很兴奋，但我看配音总有种甲亢憨憨的感觉。校长昂热配的总感觉还不够老成，我个人认为昂热是那种旧时代的残党，怀着复仇的烈火活到现在，声音怎么着也该有一点宝刀尚在的沧桑嘶哑，结果配音像个四十岁上下的不服老中年大叔。有时候路明非自己的内心吐槽戏安排的也不是很好，有时候还容易打乱节奏，当然还有芬格尔等等人的配音我也觉得有问题，就不多吐槽了。

然后就是在长江三峡的猎龙行动，卡塞尔学院的载具不能说是科技感满满，那也是未来科技了，原作的一个水面船只，到了动画里直接摇身一变变成了个星舰一样的舰船，这多少有点离谱了，且不论你这艘舰船能不能像正常星舰一样拉升高度摆脱龙侍追击，就说在龙侍在追击的时候舰船全程没有拉升高度，可是你这么个飞行器也不像地效飞行的气动外形啊，就算你这艘船真的是个浮空战舰，那你就不能拉升高度摆脱顺便扔点深水炸弹啥的表示表示？总而言之，这段剧情非常的离谱，不论是在还原还是表演上。当初我差不多就是看到这里就看不下去了，观感崩的太快了。如果是小说漫画都没有看过的纯新人，我认为你可以去试一试，毕竟原作和画的还不错的漫画你都没看过，就这么一个相对不是很还原的动画应该能满足新人天马行空的想象力。

现在想想还是很可惜，尽管第一部问题有点多，但是在还原上确实非常的不走心，有的地方甚至非常的出戏。有人说这个画风有点老，其实我觉得还行，画风更老的《战斗妖精雪风》我都看下去了，《popo 子和 pipi 美》的画风更为抽象，就这样的都看不下去就说画风老的多少有点主观审美了。

杂谈 动画 江南 龙族
